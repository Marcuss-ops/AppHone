
# Luce - App Cristiana in Flutter

Prototipo generato per te.

## Come avviare
1. Installa Flutter: https://flutter.dev
2. Da terminale:
   flutter create luce
   cd luce
3. Sostituisci `lib/main.dart` con questo file
4. Sostituisci `pubspec.yaml` con quello fornito
5. Esegui:
   flutter pub get
   flutter run

Funziona su Android e iOS. Il Diario salva in locale con SharedPreferences.
